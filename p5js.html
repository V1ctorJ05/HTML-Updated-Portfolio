P5.Js

After working with p5.js, I found that I enjoyed it much less than HTML. The amount of code required to create an image was more than I thought it would be. Especially when compared to HTML. I also found the process of uploading it to GitHub extremely frustrating, especially since the process was different then uploading HTML files. Despite these challenges, I’m really happy with the final result. The colors turned out really nice, and I loved using simple shapes to create my self portrait. The way they came together in the design was satisfying. Overall, while the process was tough, I’m pleased with the end result even if it was a headache to understand and get right.

P5.Js Drawing

After the not-so-fun first part of using p5.js to create a self portrait, I found this second half using it as a drawing tool much easier and more enjoyable. The amount of writing and coding felt so much easier, and it was satisfying to see how the code changed how everything worked only using a couple simple lines of code to start with. I especially liked how different effects appeared depending on the movement of the mouse, which made the process feel quicker and manageable. I’m really happy with the result, especially the colors and the chaotic style I aimed for as a form of stress relief. Using the original image I made in p5.js and combining it with the drawing effects felt fun and expressive.

P5.Js drawing tool

P5.Js Game

After enjoying the second part of using p5.js as a drawing tool, using it this third time to build a game was both easier and more frustrating. The amount of code wasn’t too overwhelming since I started with the teacher’s base and built off of it, which made implementing new features more manageable. I liked how just a few tweaks could completely change how the game felt and played. I’m semi-happy with the result as I really enjoy how the game functions with simple new features, though saying I struggled would be an understatement. As I had lots of issues trying to add audio even when I had Chatbot help me try to add it. And ultimately had to leave it out, which was not only disappointing but frustrating. In the end, I’m glad the game works and that all the new features I added function properly, but I do wish the audio would have worked.
